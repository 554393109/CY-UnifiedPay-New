参数规定

  • state - 通讯状态
  • code - 状态码
  • scene - 支付场景
  • trade_state - 交易状态
  • paytype - 支付方式
  • picType - 进件图片类型
  • typeId - 图像识别类型
  • DealType - 经营类型
  • LegalPersonType - 营业执照持有人类型
  • LegalPersonCertType - 法人证件类型
  • （QrCodeMchApply - 商户进件）flag_type - 商户类型
  • AccountType - 结算账户类型
  • IdCardType - 结算账户银行卡持卡人证件类型
  • PaymentChannelId - 支付通道编码

编码统一使用大写

state - 通讯状态

编码 描述
SUCCESS 成功
FAIL 失败
VER 版本过低

编码 描述
10000 成功
